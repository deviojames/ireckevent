# Event application for Ireckonu assignment
- use MirageJS to mock API server.
- Redux
  - redux-thunk
  - redux-persist
- styled-components
- react-navigation

## Setup
- Clone project
- Run
```
$ npm i
```
- Install Pods
```
$ cd ios/
$ pod install
```

## Run Project
```
iOS
$ npx react-native run-ios 
Android
$ npx react-native run-android 
```

## Test
```
$ npm test
```
- Test coverage
```
$ npm test -- --coverage
or
$ npx jest --coverage
```
